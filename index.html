Proyectos

Verqor's Farming Tycoon

Videojuego de gestión de recursos en el que administras una granja y tienes diferentes opciones de financiamiento, desastres naturales, etc, tu objetivo es obtener la mayor cantidad de dinero posible Link del repositorio de Github

Proyecto de Sistemas Multi-Agente

Este es una simulación de un sistema de tránsito utilizando agentes múltiples, cada agente tiene características y comportamientos específicos que son simulados en utilizando el framework MESA en python, esta simulación además tiene una representación en 3D usando Unity, los modelos de los autos fueron realizados por nosotros usando ProBuilder y todo el movimiento está hecho con proyecciones matemáticas propias Link del repositorio de Github

Todas Brillamos

Este proyecto fue consistió en una aplicación móvil hecha en Android Studio usando Kotlin que incorpora una tienda en línea, puede procesar pagos usando Paypal, y además tiene una página web que permite administrar el contenido de la aplicación Link del repositorio de Github

Tiendita Ya

Este es el proyecto realizado para el HackMX6, inspirado por el proyecto de Todas Brillamos, consiste en una aplicación móvil para administrar tiendas de abarrotes, incorpora el uso de inteligencias artificiales para el reconocimiento de letra manuscrita, permitiendo a los usuarios registrar de la misma manera que siempre lo han hecho pero con los beneficios de digitalización Link del repositorio de Github
